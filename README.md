# 🩺 Breast Cancer Prediction using KNN

A machine learning project that predicts whether a breast tumor is **Benign** or **Malignant** using the K-Nearest Neighbors (KNN) algorithm.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- Joblib

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
